# Weather App

A responsive weather dashboard built with HTML, CSS, and JavaScript. The app uses live data from the Open-Meteo API to show current weather, hourly forecasts, and a 7-day outlook in a clean, mobile-friendly interface.

## About

I built this project to practice working with external APIs, asynchronous JavaScript, DOM updates, and responsive UI design. The goal was to create a simple but polished weather app that feels useful, easy to navigate, and suitable for an entry-level frontend portfolio.

## Features

- Search weather by city
- Use current location with browser geolocation
- Switch between Celsius and Fahrenheit
- View current temperature, humidity, wind, sunrise, and sunset
- Check hourly and 7-day forecast data
- Save recent city searches in local storage
- Responsive layout for desktop and mobile screens

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Open-Meteo API
- GitHub Pages
